钢铁侠是怎样炼成的：一段给人启示的英雄传奇

发表于 2019-08-07 | 分类于 读书 ， 传记

被称为硅谷钢铁侠的埃隆·马斯克（Elon Musk），
是继乔布斯之后，硅谷最受瞩目的创业家和实业家。
也是电影《钢铁侠》的原型。
取得了很多令人艳羡的成就：

  • 拥有电动汽车标杆品牌特斯拉，
  • 创立了世界上最稳定的火箭运营商Space X，
  • 国际支付公司PayPal的联合创始人，
  • 美国最大太阳能公司Solar City最大股东

而普通人能取得以上任何一项成就足以称为成功。

本文就来探讨这些成就背后的精彩故事以及值得思考和学习的地方。

少年学霸

马斯克出生于南非，一个种族冲突非常激烈的国家。
在他小时候父母离异，先跟随母亲生活了几年，12岁后又与父亲生活到18岁。

如果抱着当下流行的“原生家庭决定未来”这样的观点来看（这个观点其实并不新鲜，可以说是弗洛伊德的童年决定论通俗版，但事实证明与之相反的阿德勒理论更有现实意义），马斯克不走上违法犯罪的歧途已经算奇迹了。
但其实很多伟大的人都是在单亲家庭成长的：孔子、孟子、亚里士多德、岳飞、牛顿、奥巴马。

但马斯克并没有在不幸的童年中过度沮丧，而是不停地阅读学习，13岁开发了一款名为“炸弹”（blastar）的游戏并赚到了500美金。

对于马斯克来说，父母离异远不是最惨的。

上初中的时候遭遇校园霸凌长达三四年之久，严重的一次被黑社会打晕送至医院住院一周。
甚至殴打他朋友，直到那个朋友答应不再和马斯克一起玩为止。

17岁时偶然继承了母亲的国籍，只身前往加拿大求学，18岁时进入皇后大学（因为这里美女多）。
变态心理学课程获得了98分，通过与教授争辩争取到了扣掉的2分。
2年后转学到常春藤名校宾夕法尼亚大学，主修双学位——经济学和物理学。
同学对他的评价是“一旦埃隆想了解某个事物，他会投入比别人多得多的精力。这是埃隆和其他人的不同之处。”

很多时候我们看到的学霸、成功人士觉得他们天赋异禀，但很可能只是他们把普通人用来玩乐休息的时间花在了自己觉得有价值的事情上而已。

大学毕业后考虑进入游戏行业，但最终放弃，因为觉得即使做出了非常出色的电脑游戏，也无法对世界产生很大影响。
通过对未来世界的思考，认为互联网、可再生能源和太空探索这三个领域将会对世界发展起到巨大作用。
当时打算同时进军这些领域，但实际上可以看到先进入的是互联网行业。

这种这种视野和格局以及对未来的前瞻性太令人羡慕，注意他当时只是一个刚毕业的大学生。
而对于普通人在做职业规划的时候是不是也有借鉴意义？
很多人迷茫的时候喜欢提找回初心，但这其实就是一种固定思维，初心跟不上事物的发展未必是对的、好的。
我比较喜欢的一种思维方式就是“终局思维”。
例如2007年马云在阿里达到100亿美元市值地时候，做出未来十年地战略决策：建设一个电子商务生态系统，而不是只盯着当时最赚钱的淘宝业务，才有了十年后的数千亿美元市值。
简单地理解，终局思维就是立足多年以后的“终局”，逆向思考来决定眼下最重要的事情。
这个思维方式也一直帮助我走到了今天，从不受公司待见地默默无闻的程序员到课程讲师、团队负责人、技术书作者。
关于我的故事以后有机会将会在公zhong号“web学习社”中分享。

第一桶金

马斯克本打算在斯坦福大学攻读材料学和物理学博士，但在大学待了两天就退学投身互联网了。
和弟弟在硅谷创办了一个黄页网站（可以理解为简单版的大众点评）。
当时马斯克一个人搞定了系统开发，而他的弟弟负责上门推销。
他几乎从来没有离开过办公室，累了就睡在办公室的睡袋里。
并且告诉公司员工，谁早上到了公司就把他踢醒，然后他再继续工作。
由于得到了风险投资以及与传媒集团合作，公司逐渐获得成功。
最终以3.07亿美元的价格被收购，马斯克获得了2200万，当时27岁。

这一段艰苦的创业经历充分地体现了马斯克做事地专注与毅力。
专注力其实真的太宝贵了。
专注力本身就是进入心流状态的必要条件，而只有进入心流状态才能高效地工作、学习，从而取得成就。
但是现在这个注意力经济的社会，有太多的产品想方设法分散人地注意力并让人沉迷其中。
打开游戏一玩，短视频一刷，网剧一看，其中一件任何事情只要一开始很可能一晚上就过去了。
工作时也有各种网站广告、聊天信息分散注意力。
更可悲的是这些行为不但浪费了宝贵时间，还让人成了商家赚钱的工具。
同时从另一个角度来看，对于处理某些事情，我们如果摆脱干扰和杂念，专门划分一段时间废寝忘食的处理，而不是进行拆分和延后，是否效果更好呢？
比如常常听说的封闭式开发~我们是否可以对自己也实行封闭式学习呢？
毅力的重要性就更不用说了，让事情在时间的长河中形成势能，而且这种势能可能会是牢不可破的护城河。
比如罗辑思维每天60秒的语音，这种坚持就会形成品牌效应，而且后来再有人做类似的事情也无法在时间维度上超越它。

大多数人理性的人有了上亿的钱，突然成了暴发户，肯定会拿来做投资理财，然后提升一些生活。
那马斯克当时是怎么做的呢？

连续创业

第一步，把自己打造成硅谷的花花公子（焦点人物），让媒体来报导他，其中包括：

  • 购买了一套167平米的公寓。
  • 买了一辆价值100万美元的迈凯伦F1跑车，全世界仅62辆。
  • 买了一架小型螺旋桨飞机并学习开飞机。

同时马斯克第二个月便创建了一家新公司X.com，并将剩下的大部份钱都投了进去。

与其说这是暴发户挥金如土，而实际上是一种建立个人知名度与扩展人脉的最快手段。
并同时宣传了自己的新公司。
很多时候产品的营销可能会比产品本身质量更重要，比如创造销售奇迹的脑白金。
所以这里可以看出马斯克并不是一个只对技术感兴趣的宅男，很早就意识到了知名度、影响力的重要性。

这就是几年后与彼得蒂尔创立的公司合并而成的PayPal。
最终以15亿美元价格被收购，马斯克完税后获得1.8亿美元，成为亿万富翁。

大多数人在这个时候基本上就可以过上舒服的退休生活了，最多搞个慈善基金，写几本书。

但马斯克的进取心完全没有衰减。

在互联网领域获得成功后，他转向太空探索领域，组建火星生命基金会。
从俄罗斯采购洲际导弹改装成火箭的想法失败后，花几个月时间研究航天工业及其背后的物理原理，详细计算了制造火箭的成本，然后成立Space X公司决定从零开始建造更廉价的火箭。

艰难时刻

造火箭的难度当然是巨大的，尤其是他承诺要造出更便宜、载重更多、的可回收火箭时。

  • 2005年11月，第一次发射因氧气罐阀门故障终止。
  • 2006年3月，飞行不到25秒便失控坠落地面。
  • 2007年3月，升空4分钟后直接爆炸。

此时的 Space X 就像是一个无底洞，已经快要把马斯克的钱包掏空了。

同时在2004年他入股特斯拉，2005年造出第一辆原型车，虽然收到很多订单，但很快发现售价8.5万美元的Roadster成本却要20万美元，于是亲自参与研发削减成本。

2008年经济大萧条时是他最艰难的时候，时刻处于破产的边缘。

  • 信用接近破产。网上舆论开始讨论特斯拉倒闭时间，甚至一些离职员工开始在网上曝光说特斯拉根本就是在骗人。最多的时候一天出现过50篇关于讨论特斯拉如何灭亡的文章。
  • 经济接近破产。Space X 第三次发射依然失败。
  • 生活接近崩溃。妻子当时也在和他闹离婚。

钢铁侠诞生

英雄的故事虽然各不相同，但精彩人生总是大起大落。

2008年9月，Space X 第四次发射成功，赢得了NASA 16亿美元的订单。

同时在圣诞节前夕特斯拉也获得新一轮融资。而当时马斯克只剩下几十万美元，甚至无法第二天给员工支付薪水。

如今 Space X 已经称为最稳定最廉价的太空服务商之一，特斯拉也成为了电动汽车行业的标杆。

当然马斯克的太阳城、超级高铁计划也在稳步推进。

坚强、好学、专注、坚持，能做到这些，也必将拥有钢铁侠一般英雄的人生~
